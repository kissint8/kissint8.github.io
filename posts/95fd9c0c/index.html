神威日记1-Macbook登录

最近在摸索国产神威，由于笔者用的是macbook，登录踩了一点小坑。

优化中字版

1. 部署VPN客户端

我们拟采用docker来启动神威的VPN客户端，并将容器的指定端口转发映射到宿主机的端口，用作ssh登录的代理接口。

  1. 先确保安装docker。

  2. 创建docker脚本，并初始化。

    • <forward-port>：映射到宿主机的端口，可任意填写，尽量避开常用端口段(笔者用了1088-1099)。

      • 备注：后续如果需要多开VPN，只需要另增容器，转发不同的端口即可。

    • <container_name>：容器命名，后续许多docker命令的操作标识，建议简短而明确。

    • <vpn_user>和<vpn_passwd>：VPN的用户名和密码。

    • <vpn-ip:vpn-port>：VPN的server_ip:server_port。

  3. 创建并启动容器。

    初次运行需要先创建容器：

    成功运行，你将看到屏幕中打印出的容器id，此后无需再运行此脚本。

  4. 检查容器状态。

    • docker ps -a 可以罗列出当前宿主机运行的全部容器及其运行状态。

    • docker logs <container_name> 可以查看容器的运行输出，在此能看到VPN端的登录过程以及运行状态。

      • 另一个直观的方法是使用curl --proxy socks5://127.0.0.1:<forward-port> http://checkip.amazonaws.com，若能回显你的ip地址，则说明代理已通。

    • docker <start|stop|restart|rm> <container_name> 当客户端异常时，可以考虑重启或者删除重建。

简洁英版

Create VPN

  1. copy topsap-docker.sh
  2. modify the <username> <password> and <ip:port> as the VPN settings.
  3. modify the <container_name> as an explicit name (e.g. sunway_vpn).
  4. Be sure that you have installed the Docker.
  5. exc bash topsap-docker_copy.sh.

If you run above script for first time, you’ll see the download process for waldon/topsap:arm64 and get a for your vpn-container.
Otherwise, you’ll only get the when container is created and running successfully.

Check Container

  1. You can check container status via docker ps -a. You can find up any times for your container in STATUS.
  2. You can check vpn logs via docker logs <container_name>. If you can’t find “Login Success!“, there must have been a big Bang somewhere in your preference!
  3. You can check vpn adress via curl --proxy socks5://127.0.0.1:1088 http://checkip.amazonaws.com. If all is right, you can get the eyour native ip address.

SSH over Proxy

  1. you can directly use -o and ProxyCommand as the following:
    ssh -o ProxyCommand="nc -x 127.0.0.1:1088 %h %p" <username>@<ip>
    you dont need modify %h and %p.

  2. Alternatively, you can add to ~/.ssh/config as following:

Modify the <server_name> as an explicit short name.
Modify the <ip> and <username> as the Sunway Server IP address and username.
